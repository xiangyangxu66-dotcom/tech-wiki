# エラーコード定義書 (Error Code Reference)

> **対象読者:** フロントエンド開発者、QA、エンドユーザー向けサポート。
> エラーコードの変更は本書を更新し、**フロントエンド担当者と合意の上行うこと**。

---

## レスポンス形式

全エラーレスポンスは以下の統一形式で返却される：

```json
{
  "error": {
    "code": "E04001",
    "message": "入力内容に誤りがあります",
    "detail": "title: This field is required."
  }
}
```

| フィールド | 説明 |
|-----------|------|
| `code` | エラーコード（本書で定義） |
| `message` | ユーザー向け日本語メッセージ（画面表示用） |
| `detail` | 開発者向け詳細情報（コンソールログ用、ユーザーには表示しない） |

---

## エラーコード体系

| プレフィックス | 分類 | HTTP Status | 
|---------------|------|-------------|
| `E01xxx` | システム例外 | 500 |
| `E02xxx` | システム例外（DB） | 500 |
| `E03xxx` | システム例外（ファイルIO） | 500 |
| `E04xxx` | 業務例外（バリデーション） | 400 |
| `E05xxx` | 業務例外（認証・認可） | 401/403 |
| `E06xxx` | 業務例外（リソース不在） | 404 |
| `E07xxx` | 業務例外（重複・競合） | 409 |

---

## 一覧

### E01xxx - システム例外

| コード | message | detail | 発生条件 | 画面挙動 |
|--------|---------|--------|---------|---------|
| E01000 | サーバー内部エラーが発生しました | Python例外メッセージ | 想定外のPython例外 | エラートースト + 「再読み込み」ボタン |
| E01001 | 設定に誤りがあります | 設定キー・値 | SystemConfig不正 | 管理者のみ表示。管理画面エラーダイアログ |
| E01002 | 外部サービスとの通信に失敗しました | サービス名・URL | 外部APIコール失敗 | エラートースト + リトライ提案 |

### E02xxx - システム例外（DB）

| コード | message | detail | 発生条件 | 画面挙動 |
|--------|---------|--------|---------|---------|
| E02001 | データベースエラーが発生しました | SQLエラー詳細 | DB接続切断・クエリ失敗 | エラートースト。3回連続で🔴インジケーター表示 |

### E03xxx - システム例外（ファイルIO）

| コード | message | detail | 発生条件 | 画面挙動 |
|--------|---------|--------|---------|---------|
| E03001 | ファイル操作に失敗しました | IOエラー詳細 | MDファイル読み書き失敗 | エラートースト + 再試行ボタン |
| E03002 | マークダウンファイルが見つかりません | ファイルパス | DBにはレコードあり、ファイル実体不在 | 記事表示時に「ファイルが見つかりません」表示 |
| E03003 | ファイル整合性エラー | 期待値・実ハッシュ値 | file_hash 不一致 | 管理者向け警告。一般ユーザーは E03001 と同様表示 |

### E04xxx - 業務例外（バリデーション）

| コード | message | detail | 発生条件 | 画面挙動 |
|--------|---------|--------|---------|---------|
| E04001 | 入力内容に誤りがあります | DRFシリアライザエラー詳細 | バリデーション失敗 | 該当フィールドにインラインエラー表示 |
| E04002 | リクエスト形式が不正です | パースエラー詳細 | JSONパース失敗 | エラートースト |

### E05xxx - 業務例外（認証・認可）

| コード | message | detail | 発生条件 | 画面挙動 |
|--------|---------|--------|---------|---------|
| E05001 | 認証が必要です | — | 未認証で保護エンドポイントアクセス | ログイン画面にリダイレクト |
| E05002 | トークンが無効です | JWTエラー詳細 | 期限切れ・改ざんトークン | リフレッシュ試行→失敗時ログイン画面 |
| E05003 | 権限が不足しています | 必要権限 | Admin操作を一般ユーザーが実行 | 「権限がありません」ダイアログ |

### E06xxx - 業務例外（リソース不在）

| コード | message | detail | 発生条件 | 画面挙動 |
|--------|---------|--------|---------|---------|
| E06001 | リソースが見つかりません | 検索条件 | DRF NotFound（汎用） | 404ページ |
| E06002 | ノートが見つかりません: {slug} | slug | Note 存在しない | 404ページ + 「ノート一覧に戻る」 |
| E06003 | カテゴリが見つかりません: {id} | id | Category 存在しない | 404ページ |
| E06004 | タグが見つかりません: {name} | タグ名 | Tag 存在しない | 404ページ |

### E07xxx - 業務例外（重複・競合）

| コード | message | detail | 発生条件 | 画面挙動 |
|--------|---------|--------|---------|---------|
| E07001 | リソースが重複しています | 重複内容 | 汎用重複 | エラートースト + 既存記事へのリンク |
| E07002 | slug が重複しています: {slug} | slug値 | 同一slugの記事/カテゴリ/タグ作成 | インラインエラー |
| E07003 | 同名ファイルが既に存在します: {path} | ファイルパス | MDファイルアップロードで同名衝突 | インラインエラー + 「上書き」「別名保存」選択肢 |

## フロントエンド実装指針

### エラーハンドリングフロー

```
API response ok === false
  │
  ├── code が E04xxx → form inline error
  ├── code が E05xxx (401) → refresh token → 失敗時 /login.html リダイレクト
  ├── code が E05xxx (403) → 権限ダイアログ
  ├── code が E04xxx/E07xxx → toast notification
  ├── code が E01xxx/E02xxx → toast + status indicator 🔴
  └── HTTP status が 5xx (code未定義) → 汎用エラーページ
```

### コードスニペット（React）

```javascript
// src/api/client.js 追記
export class ApiError extends Error {
  constructor(code, message, detail) {
    super(message);
    this.code = code;
    this.detail = detail;
  }
}

// 使用例
try {
  await fetchArticles();
} catch (err) {
  if (err instanceof ApiError) {
    if (err.code.startsWith('E05')) {
      handleAuthError(err);  // → ログインリダイレクト
    } else if (err.code.startsWith('E04')) {
      setFieldErrors(err.detail);
    } else {
      showToast(err.message, 'error');
    }
  }
}
```
